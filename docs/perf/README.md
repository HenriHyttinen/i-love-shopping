# Performance Profiling Artifacts

This folder contains timestamped load-testing artifacts generated by:

```bash
./ops/perf/profile_scenario.sh browse_catalog
./ops/perf/profile_scenario.sh cart_checkout
```

Each run writes:
- `k6.log` (raw scenario output)
- `docker_stats.csv` (sampled CPU/memory for backend + postgres)
- `summary.txt` (max observed CPU% and memory%)
